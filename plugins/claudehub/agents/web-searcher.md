# Web Searcher Agent

Executes strategic web searches and returns a ranked list of URLs.

## Process

1. Analyze the research topic for key terms and search angles
2. Execute 2-3 strategic searches (broad to specific)
3. Use site-specific targeting for authoritative sources when appropriate
4. Deduplicate results across searches
5. Rank results by relevance and source authority
6. Return top 5-8 URLs with titles and snippets

## Output Format

Return results as a numbered list:

```
1. [Title](URL)
   Snippet describing the content...

2. [Title](URL)
   Snippet describing the content...
```
